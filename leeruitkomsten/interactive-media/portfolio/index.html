Portfolio

Introductie

De portfolio kregen we als ons eerste opdracht geintroduceerd, dit zou dienen als development leeruitkomst maar ook als een website om op een interactieve wijze je andere leeruitkomsten aan te tonen. Ik ben in eerste instantie begonnen met het designen van mijn portfolio en ben vervolgens aan de slag gegaan met het omzetten in code (HTML, CSS en JS). Al mijn code is terug te vinden op mijn GitHub repository.

GitHub

Structuur

Ik heb voor de opbouw van mijn portfolio HTML gebruikt, hier heb ik altijd al wat ervaring mee gehad en dit was natuurlijk niet de moeilijkste taak. Ik heb wel geprobeerd om mijzelf tijdens het project te verbeteren in mijn HTML en zo min mogelijk code te schrijven ervoor. Los hiervan heb ik tijdens het project zelf nog mijn structuurwijze aangepast en ben overgegaan op verschillende mappen voor iedere leeruitkomst om het zo overzichtelijker voor mezelf te maken.

Opmaak

Alle opmaak is geschreven in CSS. Ik was nog niet heel ervaren met CSS en heb geprobeerd om alles zo efficient mogelijk te schrijven. Ik heb geen gebruik gemaakt van CSS libraries, dus het is allemaal pure CSS van scratch geschreven. Ook heb ik hier weer tijdens het project zelf mezelf verbeterd en ben ik efficiÃ«nter aan de slag gegaan met CSS. Zo ben ik op de laatste pagina's veel minder regels gaan schrijven.

Interactie

Ik heb in mijn portfolio meerdere keren animaties gebruikt, dit omdat ik vind dat animaties een nieuwe vorm van smoothness geeft aan je pagina. Ik heb verschillende animaties gebruikt, eentje heb ik zelf gemaakt in CSS, deze wordt voornamelijk gebruikt om alle variabelen een fade-in met een kleine rotation te geven. Ook heb ik de AOS JavaScript library gebruikt om stukken tekst in te laten faden als je ze voorbij scroll. Tenslotte heb ik nog zelf een menu animatie in JS gemaakt als de pagina verkleind wordt, zodat je alsnog kan navigeren.

Wat heb ik geleerd?

Ik heb geleerd hoe je basis-JavaScript toepast in een HTML/CSS project en wat je er nou eigenlijk allemaal mee kan. Ik heb hier voorheen nog nooit met JavaScript gewerkt. Ook heb ik geleerd hoe je d.m.v. keyframes verschillende animaties binnen CSS zelf kan maken en deze kan toepassen op bijv. tekst. Ook heb ik geleerd hoe je JavaScript libraries toevoegt en toepast aan je project. Op deze manier is animaties maken ook erg eenvoudig geworden. Ook heb ik een complete eigen menu gemaakt met JavaScript, wat voorheen ondenkbaar was voor mij.

Feedback

Ik heb ontzettend veel feedback ontvangen, waar ik ook veel mee gedaan heb. Ik heb aan de development zijde veel feedback gehad op de responsiveness, dit was op het begin heel slecht en ik heb het naarmate ik mijn project heb voortgezet ook verbeterd. Ook heb ik meegekregen dat animaties een goede toevoeging zijn aan de site, en het meer leven geven, dat heb ik vervolgens ook toegepast. Van de leraren heb ik meegekregen dat je nooit genoeg werk in je portfolio kan hebben, dus op gegeven momenten moest ik meer op het werk dat ik in mijn portfolio liet zien focussen dan op development zelf.
