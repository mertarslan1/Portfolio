Rock Paper Scissors

Voor mijn eerste Proof of Concept heb ik in JavaScript een rock paper and scissors spel gemaakt. Aangezien ik nog (bijna) geen ervaring met JavaScript had, heb ik dit d.m.v. een YouTube-tutorial gemaakt. Ik heb in eerste instantie de tutorial stap voor stap gekopïeerd en vervolgens mijn eigen huisstijl (lettertypen, css) toegepast.
